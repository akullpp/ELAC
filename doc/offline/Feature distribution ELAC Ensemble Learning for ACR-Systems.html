Extracting feature distribution

General

In ELAC we included a standalone tool to extract the total and average distribution of features from a corpus. These can be used to compare corpora and find explanations for their differing results e.g. in your anaphora and coreference resolution routine.

How to use

From the demo CLI select the appropriate option Feature Distribution and in the following prompt choose between the training and testing corpora, depending from which you want to extract the features. Note that the paths to both corpora must be set correctly in the config.xml file.
After the process is finished, you’ll find a file in the ResultOutputDir (specified in config.xml) with the total/average distribution of each feature in the selected corpus.
