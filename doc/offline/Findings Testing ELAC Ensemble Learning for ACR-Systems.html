ELAC: Ensemble Learning for ACR-Systems

Testing

Posted on 1. August 2010 [Edit] Leave a comment (0)

Machine Learning results (not domain-specific)

Using our meta learning approach to train machine learning classifiers on meta data about the confidence and trustworthiness under certain feature distributions, we achieved the following results:

Overall results (F-Score)

Comparison of ML, baseline and average results (f-score)

We chose the ACR system achieving the best F-Score as our baseline (e.g. BART) and compared the results scored by the usage of different machine learning classifiers. Although we couldn’t beat BART’s performance, the average result of the different ACR systems should also be considered as their quite weak performance (this is especially true for JavaRap) might have lead to confusion for the machine learning classifiers.

Comparing the average performance achieved by all three ACR systems used, the ML approach clearly outperforms the average of the single systems.

Precision

Comparison of ML, baseline and average results (precision)

Using machine learning, the precision improved (depending on the classifier used), so this could be seen as an approach to filter out bad guesses by ACR systems (which was originally intended). This should be especially useful for combining preferably complementary ACR systems, which tend to achieve a high performance only for certain phenomena.

Recall

Comparison of ML, baseline and average results (recall)

Note: Lingpipe outperforms BART regarding recall results, but as BART scores higher in F-Score results, we decided to choose it as our baseline for all the measurements to ensure comparability.

Comparing the baseline to our machine learning results, the reason for mediocre F-Score results seems to be quite obvious: it was not possible to outperform the baseline results, although the machine learning approach scored slightly higher than the average recall results.

Comment

Our guess is, that BART being quite robust across all feature combinations, the other systems (being specialized on one specific kind of anaphora and coreference resolution) couldn’t contribute really “new” knowledge to the already quite reasonable results achieved by BART. Combining several specialized systems could be an interesting topic for further investigation and research.
