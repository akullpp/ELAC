ELAC: Ensemble Learning for Anaphora- and Coreference-Resolution-Systems

Abstract

ELAC's core functionality is to utilize several Anaphora- and Coreference-Resolution-Systems (ACRS) in order to use their predictions for training an ensemble of (meta-)machine-learning algorithms.
This ensemble classifies the ACRS predictions on a test corpus as either trustworthy or untrustworthy and tries to achieve a better recall/precision than a single ACRS.
There are several non-core functionalities as well:

  • A tool which extracts the feature distribution in a corpus.
  • A parser which transforms Salsa/Tiger format to an inline notation which can be processed by MMAX2.
  • An ablation testing tool which determines the best feature sequence.
